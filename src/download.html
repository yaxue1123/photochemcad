Download PhotochemCAD™

  1. Download and un-ZIP the PhotochemCAD™ program files.
  2. Double-cick photochemcad 3.exe file.
  3. Open the database tab (File => Open => Database).
  4. Locate the unzipped folder and select the file named 2018_03 PCAD3.db.

Operating System requirements

  • PhotochemCAD™ runs under all Windows operating systems.
  • There is no Macintosh version of PhotochemCAD™.
